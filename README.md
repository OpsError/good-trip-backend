# Backend for "good-trip"

## Стек:
- Node.js
- Express.js
- Multer

## Планы по доработке:
- Регистрация (создание папки пользователя для хранения картинок)
- Авторизация (помещение токена в куки)
- get-запрос на получение картинок
- Добавление/удаление "карточек"
